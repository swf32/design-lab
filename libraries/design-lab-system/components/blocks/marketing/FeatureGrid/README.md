# Feature Grid

Centered marketing section with an optional eyebrow, headline, description, and a responsive grid of icon feature cards.

## Props

- `eyebrow` — optional uppercase label above the headline.
- `title` — section heading.
- `description` — supporting copy under the headline.
- `headingId` — optional id for `aria-labelledby`.
- `items` — array of `{ icon, title, description }` cards rendered in the grid.
