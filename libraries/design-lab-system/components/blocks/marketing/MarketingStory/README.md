# Marketing Story

Two-column marketing section with a media panel on the left and supporting copy on the right.

## Slots

- `media` — image or other media rendered inside the bordered figure.
- `description` — narrative copy under the headline.
- `actions` — optional call-to-action controls below the copy.

## Props

- `eyebrow` — optional uppercase label above the headline.
- `title` — section heading.
- `headingId` — optional id for `aria-labelledby`.
