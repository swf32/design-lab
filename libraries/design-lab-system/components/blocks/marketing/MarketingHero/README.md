# Marketing Hero

Top hero section for marketing landing Pages: headline + description, primary actions, an optional sign-in status line, and a media panel.

## Slots

- `chip` — optional compact label above the headline (for example “Now in review”).
- `description` — hero copy rendered under the headline.
- `actions` — the main call-to-action buttons.
- `status` — optional status metadata line.
- `media` — a React node rendered inside the bordered media panel (typically an `<img />`).
