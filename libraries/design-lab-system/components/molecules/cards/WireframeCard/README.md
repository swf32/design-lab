# Wireframe Card

Catalog identity for a discovered page-level Wireframe. The card renders the real adjacent Wireframe screen through `WireframeScreenPreview`, then places only the page name and lifecycle over a quiet bottom gradient.

The action's accessible name retains the description plus layout, saved-state, and transition counts without making the visual card noisy. Use `selected` only when the route continues to show the catalog; fullscreen review routes normally replace the module shell.
