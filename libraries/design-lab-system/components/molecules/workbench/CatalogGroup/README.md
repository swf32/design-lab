# Catalog Group

Semantic grouping for cards in Components, Assets, Wireframes, and Pages catalogs. Its header uses
one shared hierarchy and a `spacing.3` inline inset, aligning the label optically with content inside
rounded catalog cards instead of their outer border edge.

Omit `title` when the current folder already supplies the only useful category label. The root still
remains a Component boundary and owns mobile separation between adjacent groups.
